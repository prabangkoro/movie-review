# Books Blog
A simple book catalog for me.

[Books Blog](https://books.babangcreator.com/)

# Run on Local
```
npm start
```

# Build Project
```
npm run build
```
output folder will be on `/blog/.vuepress/dist`

# Config file
location: `/blog/.vuepress/config.js`
